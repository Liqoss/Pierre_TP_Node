Chaque champs texte du QCM dans la page ./Add doit être validé en appuyant sur la touche 'Entrée'. 
Une fois appuyé sur la touche, pour modifier avant envoi il faut raffraichir la page et recommencer la saisie.

Le back utilise le port 5000.
Le front utilise le port 3000.

Technos utilisés : 
- React
- Node.js
- Express
- Multer
- JsonWebToken
- bcrypt
- MongoDB
